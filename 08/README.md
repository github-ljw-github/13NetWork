# teddy
一个简易HTTP命令行下载器

### 功能：
* 命令行下载器
* 提供进度条显示
* 支持断点续传
 
### 编程环境
* Ubuntu16.04
* [GEC6818开发板](https://weidian.com/item.html?itemID=2167617741)

### 技术点
* 嵌入式C语言
* Linux文件IO
* TCP/IP协议解析，经典socket编程
* HTTP协议解析
